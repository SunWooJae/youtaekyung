태경이에게

안녕 태경아, 선우재야. 잘 지내? 헤어지고 원래는 이 선물 안 주려고 했어. 이미 끝난 사이라고 생각했으니까. 그래도 매일매일 밤 10시부터 12시까지 연등하면서 하던 거. 우리 싸운 목요일부터 안 했었어 내가. 그냥 주면 안되겠다~라고 마음 접어뒀었는데, 무의식적으로 계속 고민하고 있었나봐. 일요일 밤에 '알고있지만,' 이라는 드라마를 생활관에서 다 같이 보다가, '우리 사이'라는 단어가 나왔는데, 고그 듣고, 그래 내가 태경이랑 갈라졌지만, 선물도 못 줄 사이인가? 그래도 예전엔 좋은 친구였고, 내가 좋아한 사람으로, 크리스마스에 선물 줘서 조금이라도 기분 좋게 만들 수 있으면 됐다. 라는 심정으로 이 선물 준비하고 마무리했어. 그래도 최종 단계까지 어떻게 깻네?? 대단해~~~피하기 게임 안 어려웠어..? 그거 어려우라고 만든건데 ㅋㅋ 그게 난 제일 어렵더라. 우클릭 후 '페이지 소스 보기' 누르거나, '개발자 도구'로 내가 코딩한 것들 볼 수 있어. 12월 초부터 코딩 시작했어. html, CSS, JavaScript 다 공부해야 웹사이트를 만들 수 있어서 좀 걸렸다 하하. 당시 사귈 때 너한테 크리스마스 선물로 뭘 줄까 엄청 고민했어. 가장 중요하게 생각한 것은, 좋은 "경험"을 줘야겠다는 것이였어. 단순한 물리적 선물을 주기 보단, 선물을 기대하고, 여는 과정이 중요하다는 것을 느꼈어. 내가 너와 멀리 떨어진 상태에서 뭘 해줄 수 있을까아 엄청 고민한 결과! 너를 위한 웹사이트를 만들어야겠다! 결론이 났어!! 그런 선물은 태경이가 받아본 적 없을 것이고, 정말 나만 해줄 수 있는 선물이겠구나 싶었거든. 그래서 진짜 400페이지 html css JavaScript 책 선임한테 계속 빌려서 5일 안에 한 번 다 공부하고(지금도 내 앞에 펴져있어), 전체적인 선물 구조 기획, 실제로 코드로 코드 구현하고, 사진 편집해서 움직이는 움짤로 만들어서 내가 원한대로, 너한테 내 정성이 담긴 선물을 준비할 수 있었다. 20분 정도의 경험을 위해 40시간 넘는 시간을 쓰니까 중간중간 현타가 오긴 했는데 ㅋㅋㅋㅋ 정말 진심으로, 너가 좋아하는 모습 상상하면서 졸려도 코딩하고, 끙끙대면서 디버깅했어. 그래서 지난 번에 톡으로 선물하는 것의 즐거움을 느꼈다고 한거야.
헤어지면서 계획한 컨셉(우재 구출 작전)과 많이 바꼈지만, 그래도 이 선물은 꼭 주고 싶었어. 내가 뭘 원해서 주는 거 아냐. 이왕 거의 다 만든 것이고, 너가 받았을 때 조금이라도 좋아할 것 같아서 웹사이트 최종배포했어.
잘 지내 태경아 응원해. 메리 크리스마스 :)
